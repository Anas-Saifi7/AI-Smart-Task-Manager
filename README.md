# 🚀 AI Smart Task Manager

A full-stack **AI-powered Task Management System** with real-time collaboration, team management, and intelligent insights.

---

## 🌐 Live Demo

* 🔗 Frontend: https://aismarttaskmanager.vercel.app/
* 🔗 Backend: https://ai-smart-task-manager-lisc.onrender.com

---

## ✨ Features

### 🧑‍💼 User Features

* 🔐 User Authentication (Login/Register)
* 📊 Smart Dashboard with task analytics
* 📌 Create, Assign & Manage Tasks
* 📅 Deadline & Priority Management
* 🔄 Real-time Task Updates

---

### 👥 Team Management

* ➕ Invite Team Members
* ✅ Accept/Reject Requests
* 👀 View Team Details
* 💬 Real-time Team Chat (Socket.IO)

---

### 🤖 AI Features

* 🧠 AI-powered task insights
* 💬 Chat with AI Assistant
* 🎤 Voice input support

---

### ⚡ Real-Time Features

* 🔔 Live Notifications
* 🔄 Instant Task Updates
* 💬 Live Chat System

---

## 🛠️ Tech Stack

### Frontend

* ⚛️ React.js (Vite)

* 🔗 Axios

### Backend

* 🟢 Node.js
* 🚀 Express.js
* 🔌 Socket.IO
* 🔐 JWT Authentication

* 🍃 MongoDB Atlas

### Deployment

* 🌐 Vercel (Frontend)
* ☁️ Render (Backend)

## 🚀 Installation & Setup

### 1️⃣ Clone Repository

```
git clone https://github.com/your-username/AI-Smart-Task-Manager.git
cd AI-Smart-Task-Manager
```

---

### 2️⃣ Backend Setup

```
cd backend
npm install
npm start
```

---

### 3️⃣ Frontend Setup

```
cd frontend
npm install
npm run dev
```
